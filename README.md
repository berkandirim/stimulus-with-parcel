# Stimulus with Parcel
Stimulus example using Parcel bundler.

- Install Parcel
```
$ yarn global add parcel-bundler
```

- Install dependencies
```
$ yarn install
```

- Start app
```
$ yarn start
```

Search for a planet in Star Wars universe. Then click on the name to get the planet info.
